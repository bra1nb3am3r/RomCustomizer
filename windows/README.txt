Any use of ROMCustomizer is done at your own  risk.

First unzip the file.
Then doubleclick RomCustomizer.bat

In order to flash fonts, bootanis etc.
Find the correct folder such as FontGoesHERE or BootAniGoesHERE or SplashGoesHere
and place them in there. This will allow the program to find and use them.

Follow the instructions in the program.
